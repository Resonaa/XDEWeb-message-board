## Development

- Install dependencies:

  ```sh
  npm i
  ```

- Setup database:

  ```sh
  npm run setup
  ```

- Build the app:

  ```sh
  npm run build
  ```

- Start dev server:
  ```sh
  npm run dev
  ```
